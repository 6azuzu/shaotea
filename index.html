Shao_tea

Идея

Создание оптимизированного по регионам, для продажи чая, посуды и аксессуаров. Продвижение культуры чая, для более глубокого ее понимания со способоностью совершить покупку.

Артефакты

  1. Концепция Shao_tea;
  2. Дорожная карта;
  3. Прототип проекта в Balsamiq;
  4. Прототип проекта в Figma;
